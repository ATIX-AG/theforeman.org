参数名字 描述
host
必填

  • Must be a Hash

必填

Validations:

  • Must be String
host[environment_id]
选填

如果已管理主机，且数值不是从主机组中继承，则需填写。

Validations:

  • Must be String
host[ip]
选填

Validations:

  • Must be String
host[ip6]
选填

IPv6 address

Validations:

  • Must be String
host[mac]
选填

Validations:

  • Must be String
host[architecture_id]
选填

如果已管理主机，且数值不是从主机组中继承，则需填写。

Validations:

  • Must be a number.
