POST /api/hosts/:host_id/interfaces
在主機上建立介面

示例

POST /api/hosts/host167/interfaces
{
  "interface": {
    "name": "test.foreman.com",
    "ip": "10.0.1.1",
    "mac": "AA:AA:AA:AA:AA:AA",
    "username": "foo",
    "password": "bar",
    "provider": "IPMI",
    "type": "UNKNOWN"
  }
}
422
{
  "error": {
    "message": "ERF13-6594 [InterfaceTypeMapper::UnknownTypeExeption]: Unknown interface type, must be one of [interface, bmc, bond, bridge]"
  }
}

參數

參數名字 描述
host_id
必填

主機的 ID 或名稱

Validations:

  • Must be String
interface
必填

介面卡資訊

Validations:

  • Must be a Hash
interface[mac]
選填 , 允許空值

MAC address of interface. Required for managed interfaces on bare metal.

Validations:

  • Must be String
interface[ip]
選填 , 允許空值

介面卡的 IP 位址

Validations:

  • Must be String
interface[type]
選填 , 允許空值

Interface type, e.g. bmc. Default is interface

Validations:

  • Must be one of: interface, bmc, bond, bridge.
interface[name]
選填 , 允許空值

Interface’s DNS name

Validations:

  • Must be String
interface[subnet_id]
選填 , 允許空值

介面卡的 Foremane 子網路 ID

Validations:

  • Must be Fixnum
interface[domain_id]
選填 , 允許空值

Foreman domain ID of interface. Required for primary interfaces on managed hosts.

Validations:

  • Must be Fixnum
interface[identifier]
選填 , 允許空值

裝置辨識子，例如 eth0 或 eth1.1

Validations:

  • Must be String
interface[managed]
選填 , 允許空值

這個介面卡是否應該透過 DHCP 和 DNS 智慧型代理伺服器管理，以及它是否應該在佈建時進行配置？

Validations:

  • Must be ‘true’ or ‘false’ or ‘1’ or ‘0’
interface[primary]
選填 , 允許空值

Should this interface be used for constructing the FQDN of the host? Each managed hosts needs to have one primary interface.

Validations:

  • Must be ‘true’ or ‘false’ or ‘1’ or ‘0’
interface[provision]
選填 , 允許空值

Should this interface be used for TFTP of PXELinux (or SSH for image-based hosts)? Each managed hosts needs to have one provision interface.

Validations:

  • Must be ‘true’ or ‘false’ or ‘1’ or ‘0’
interface[username]
選填 , 允許空值

僅適用於 BMC 介面卡。

Validations:

  • Must be String
interface[password]
選填 , 允許空值

僅適用於 BMC 介面卡。

Validations:

  • Must be String
interface[provider]
選填 , 允許空值

介面卡供應者，例如 IPMI。僅適用於 BMC 介面卡。

Validations:

  • Must be one of: IPMI.
interface[virtual]
選填 , 允許空值

別名或 VLAN 裝置

Validations:

  • Must be ‘true’ or ‘false’ or ‘1’ or ‘0’
interface[tag]
選填 , 允許空值

VLAN 標籤，這個屬性擁有比子網路 VLAN ID 更高的優先權。僅適用於虛擬介面卡。

Validations:

  • Must be String
interface[attached_to]
選填 , 允許空值

辨識此介面卡的辨識子，例如 eth1。僅適用於虛擬介面卡。

Validations:

  • Must be String
interface[mode]
選填 , 允許空值

介面的綁定模式，例如 balance-rr。僅用於綁定介面。

Validations:

  • Must be one of: balance-rr, active-backup, balance-xor, broadcast, 802.3ad, balance-tlb, balance-alb.
interface[attached_devices]
選填 , 允許空值

Identifiers of attached interfaces, e.g. ['eth1', 'eth2']. For bond interfaces those are the slaves. Only for bond and bridges interfaces.

Validations:

  • Must be an array of any type
interface[bond_options]
選填 , 允許空值

以空格區隔開的選項，例如 miimon=100。僅適用於 bond 介面卡。

Validations:

  • Must be String
interface[compute_attributes]
選填 , 允許空值

Additional compute resource specific attributes for the interface.

Validations:

  • Must be Hash
