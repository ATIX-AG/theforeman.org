Foreman Community Newsletter - March 2014: a call for testing on Foreman 1.5, a new VM snapshot plugin, Foreman 1.4.2 and news on RDO integration. Read the newsletter.

Foreman 1.4 has been released, with new features including compute profiles, image user-data (cloud-init) support, web UI enhancements, Kerberos support, a new plugin registration and extension framework, plus lots of new provider support in the smart proxy. Read more in the release notes.
